# Contributing

Thanks for improving Repo Health Lens.

1. Open an issue for scoring changes so the tradeoffs can be discussed first.
2. Keep checks explainable and based on public, observable evidence.
3. Add or update tests for behavioral changes.
4. Run the full test suite before opening a pull request.

Please do not add popularity-based scoring or rules that encourage meaningless
commits. The project should reward maintainability, not activity theater.
